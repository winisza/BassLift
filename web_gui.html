      ? Jak uruchomić backend lokalnie
      ▼
    

    

      
Wymagania: Python 3.9+, pip, ok. 4 GB RAM

      

      
1. Instalacja zależności

      
pip install fastapi uvicorn python-multipart \
    demucs librosa midiutil python-dotenv


      
2. Uruchomienie serwera

      
uvicorn server:app --reload --port 8000


      
3. Pobierz kod serwera — plik server.py jest dostępny w repozytorium GitHub.

      

      
Pierwsze uruchomienie pobiera model Demucs (~300 MB). Kolejne są szybsze.
